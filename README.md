关于 猜市Gesma

猜市Gesma 是一个中文预测市场平台（Chinese prediction market）。在这里，用户可以对未来事件的结果进行预测、买卖预测份额，并从准确的预测中获益。常见用例包括：政治、经济、科技产品发布、体育赛事等结果预测。

预测市场通过把分散信息聚合到价格中，帮助发现概率与市场共识。我们的目标是为中文用户提供一个简单、安全、透明的预测市场体验。

The Gesma codebase will be released as open source at an appropriate time, once it is ready.
